# zvDRP

### Features
- Discord display
- Discord button
- Server Rules Button
- ID
- Steamname
- Playtime
- number of players
- server icon

#### Installation
1. Drag & Drop the Script of your server
2. Open your Server.cfg / CFG Editor in TxAdmin
3. Add "ensure zvDRP" to your cfg
4. Enjoy it!
